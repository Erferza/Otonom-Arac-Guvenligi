# Otonom Araç Güvenliği

Bu proje, otonom araçlarda siber saldırı ve anomali tespiti yaparak uygun güvenli mod kararını üretmek amacıyla geliştirilmiştir.

## Proje Mimarisi

- CIC-ToN-IoT (Ağ saldırı analizi)
- CARLA (Araç davranış analizi)
- nuScenes (Görsel bağlam analizi)
- Fusion Katmanı
- Güvenli Mod Karar Mekanizması

## Kullanılan Teknolojiler

- Python
- Scikit-Learn
- Random Forest
- CARLA Simulator
- nuScenes
- CIC-ToN-IoT
